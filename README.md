## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run start

# apply migrations
$ npm run db:migrate

# generate new migration 
$ npm run db:generate -n src/migrations/:migrationName

# drop database
$ npm run db:drop
```
